# WEBSITE PPS

Website para a disciplina de padrões e projetos para sites da internet.

  - HTML (PUG)
  - CSS
  - Bower
  - NPM

### Instalação

Instalar as dependencies e devDependencies:

```sh
$ cd website-PPS
$ npm install -d
$ bower install
```

Para ambientes de produção:

```sh
$ npm install --production
```
